Privacy is the architecture

Your career is personal data.
Treat it that way.

The API, database, generated documents, and optional inference runtime live on your machine. There is no telemetry, cloud-model fallback, or remote prompt log.

  • 01
    Bounded model context

    Only evidence selected for a task reaches the local model.

  • 02
    Explicit network boundary

    Public job searches are separate, visible, user-initiated requests.

  • 03
    Portable and erasable

    Export, restore, or explicitly erase the app-owned vault.
