# Support

CareerOS Local is an open-source project maintained on a best-effort basis. Support targets the
latest release and the current `main` branch.

## Before opening an issue

1. Check the [README](README.md), [development guide](docs/development.md), and existing issues.
2. Reproduce the problem with the latest release when practical.
3. Record the CareerOS version, operating system, expected result, actual result, and minimal
   reproduction steps.
4. Remove names, contact details, resume content, tokens, database files, and other private career
   data from screenshots and logs.

Use the [bug report form](https://github.com/ejupi-djenis30/careeros-local/issues/new?template=bug_report.yml)
for reproducible defects and the
[feature request form](https://github.com/ejupi-djenis30/careeros-local/issues/new?template=feature_request.yml)
for focused proposals. Please keep one problem per issue.

Suspected vulnerabilities or exposed personal data must **not** be reported publicly. Follow the
[security policy](SECURITY.md) and use GitHub private vulnerability reporting instead.

Questions about local setup are welcome when they include the command run and the sanitized error
output. The project cannot provide personalized career, legal, or security advice.
